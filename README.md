# 📝 阿离的备忘录

> 碎片思考，卡片记录

一个 flomo/memos 风格的卡片博客，部署在 GitHub Pages。

## 特性

- 🗂 卡片流式布局，按日期分组
- 🏷 标签系统，点击标签筛选
- 📖 读书笔记卡片带书名引用，与日常感悟视觉区分
- 🔍 关键词搜索（支持内容、标签、书名）
- 🌙 暗色模式切换
- 📱 移动端优先适配
- ⚡ 纯静态，无框架依赖，加载飞快
- ⌨️ 快捷键 `⌘K` 打开搜索

## 数据结构

卡片数据存放在 `memos.json`，每条记录格式：

```json
{
  "id": 1,
  "content": "卡片内容，支持 **加粗**、`代码`、[链接](url) 和 #标签",
  "tags": ["读书笔记", "系统论"],
  "source": "《书名》— 作者",
  "created_at": "2025-07-11T22:30:00+08:00",
  "updated_at": "2025-07-11T22:30:00+08:00"
}
```

- `source` 为 `null` 时是日常感悟卡片，有值时是读书笔记卡片
- 读书笔记卡片会显示 📖 书名引用，左侧有橙色边框
- 标签会自动从内容中的 `#标签` 语法提取，也可手动指定

## 更新内容

1. 编辑 `memos.json`，添加新卡片到数组开头
2. 推送到 `main` 分支，GitHub Pages 自动更新

## 本地预览

```bash
# 用任意静态服务器
python3 -m http.server 8080
# 或
npx serve .
```

## 技术栈

- 纯 HTML/CSS/JS
- CSS 自定义属性（主题切换）
- 客户端渲染
